mdiag -a

(Moab Account Diagnostics) Synopsis

mdiag -a [accountid]

Overview

   The mdiag -a command provides detailed information about the accounts (aka projects) Moab is currently tracking.  This command also allows an administrator to verify correct throttling policies and access provided to and from other credentials. Example 1

> mdiag -a
